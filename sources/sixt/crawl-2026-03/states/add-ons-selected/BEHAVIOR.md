# Add-ons when selected

When selected, some add-ons can be added multiple times via stepper:
- Additional driver: up to 5
- Baby seat: up to 3
- Child seat: up to 3
- Toddler seat: up to 3
- Ski rack: up to 3

Other add-ons can be added once via switch/toggle.

Selected add-ons are also highlighted in the `Your booking overview` section.
